## Choosing the Tech Stack

For this project, we used **Next.js** for its server-side rendering capabilities and MDX for writing content in Markdown with React components.

### **Key Tools Used:**
- **Next.js**: Framework for React
- **MDX**: Markdown + JSX for easy content management
- **Tailwind CSS**: Utility-first styling for fast UI development
